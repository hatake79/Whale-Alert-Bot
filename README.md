# Whale Alert Bot

A simple Web3 bot that monitors large wallets and detects whale activity on Ethereum.

## Features

- Monitor whale wallets
- Detect large balances
- Generate alerts
- Expandable to transaction monitoring

## Tech Stack

Node.js  
Ethers.js  
Ethereum RPC

## Installation

git clone https://github.com/username/whale-alert-bot

npm install

## Usage

node scripts/startBot.js

## Example Output

Whale Activity Detected

Wallet: 0xabc...
Balance: 120 ETH
